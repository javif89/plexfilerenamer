Why should you rename your files for plex?

Although PLEX's media scanners are pretty robust, sometimes they cannot find your files if they do not follow a certain naming convention. PLEX specifies how to name your files in order to make it simple for the media scanner to find.

    PLEX File Renamer simplifies this task by automating the process of renaming all the files in a folder to the PLEX naming conventions

Features

  • Auto detect series name and season when folder is selected
  • Rename all files in selected folder to the plex standard
  • Rename entire series including season folders to the plex standard
